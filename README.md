## Practice Set-up
1. Install MySQL [here](https://dev.mysql.com/downloads/installer/)

2. Use this video to follow the steps to import the Sakila database (the data used for this practice) into your MySQL Workbench [here](https://www.youtube.com/watch?v=nBEAvLHYwPg)

3. After importing the dataset, use MySQL to see the results of your queries as you practice. Use this project to confirm your queries by running 'npm test'
